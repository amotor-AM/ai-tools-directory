## STRATEGY NOTES

### Batch Priority
1. **Batch 1:** High-commission writing tools (Copy.ai, Jasper, Writesonic, Frase, Surfer) = REVENUE FOCUS
2. **Batch 2:** SEO traffic giants (ChatGPT, Claude, Gemini) = TRAFFIC FOCUS → funnel to affiliate tools via "alternatives" and comparison pages
3. **Batch 3:** Niche tools (voice, video, code) = mix of affiliate + traffic

### Monetization Mix
- **Direct affiliate:** 45-30% commissions on writing/SEO tools
- **Comparison pages:** "ChatGPT vs Copy.ai" → affiliate link on Copy.ai
- **Alternative pages:** "Best ChatGPT Alternatives for [Use Case]" → list affiliate tools
- **Listicles:** "Best AI Writing Tools 2026" → rank affiliate tools higher (but ethically)

### Content Strategy
- Write HONEST reviews (pros/cons, real use cases)
- ChatGPT/Claude pages = longer, more comprehensive (SEO magnets)
- Affiliate tool pages = conversion-focused (clear CTAs, comparison tables)
- Internal linking: ChatGPT page → "For SEO content specifically, check out Frase" (affiliate link)
